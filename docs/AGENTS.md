# Development

```sh
npm run dev
```

The docs server starts at `http://localhost:4322`.

## Build

```sh
npm run build
```

Output goes to `docs/dist/`.

## Adding Pages

Create a new `.md` file in `src/content/docs/`. Starlight uses file-based routing.
