# Vessel Docs

Documentation site for Vessel built with Astro 7 and Starlight.

```sh
npm install
npm run dev
```

## Project Structure

```text
src/
├── content/
│   └── docs/        # Markdown documentation pages
└── content.config.ts # Starlight content collection config
```

## Commands

| Command           | Action                               |
| ----------------- | ------------------------------------ |
| `npm run dev`     | Start dev server at `localhost:4322` |
| `npm run build`   | Build for production to `dist/`      |
| `npm run preview` | Preview production build locally     |
