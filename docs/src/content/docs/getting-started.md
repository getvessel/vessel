Vessl turns any bare-metal VPS into your own private Vercel & Railway.

## One-Line Install

```sh
curl -fsSL https://get.vessl.dev | sh
```

## Prerequisites

- A Linux VPS (Ubuntu 22.04+, Debian 12+, or any modern distro)
- Docker installed
- A domain pointing to your server (optional, for SSL)
