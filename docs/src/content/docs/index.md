Vessl is a self-hosted PaaS that runs on any Linux VPS — deploy apps with zero-downtime, spin up databases with one click, manage S3 storage, and collaborate with your team — all from a single dashboard on your own hardware.

## Getting Started

| Guide | Description |
|---|---|
| [Installation](/getting-started/) | Deploy Vessl on any Linux VPS in under 60 seconds |
| [Deploy an App](/deployment/) | Docker, Railpack, Nixpacks, or Buildpacks — pick your strategy |
| [Create a Database](/databases/) | 15 managed engines — PostgreSQL, MySQL, Redis, Kafka, and more |

## Platform Features

| Feature | Description |
|---|---|
| [Projects](/projects/) | Organize services into environments with an interactive canvas view |
| [Storage](/storage/) | S3-compatible object storage with auto-injected credentials |

| [Serverless](/serverless/) | Write Node.js, Python, or Go functions in the browser |
| [Integrations](/integrations/) | Git providers, OAuth, Vercel import, and outgoing webhooks |
